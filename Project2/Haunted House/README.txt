This is a full fledged text based game engine that works with any any collection of text files following the provided format in "Template Room.txt". It is still in its early testing phases and comes with its number of bugs. It follows the following logic:

Each room has at most one item or action to be used or performed. Doing so will either unlock a door or provide insight into finishing the game.

Each room has at most one locked door or required action.

Once the finalRoom is reached, the game ends on using the correct item or performing the correct action.

The provided files detail a haunted house and your aspiration to acquire the treasure rumored to be hidden away inside.
